How is a HTML Document Structured?

There are 4 main pieces to each HTML document. The first is the "DOCTYPE" tag, which tells our browser that our document is HTML

The next tag, "html", just says that everything inside here is going to be the HTML we want to use on our web page.

Next is the head tag, which includes all of the information we need to have before seeing the page (extra styles, fonts, libraries etc.)

Finally we have the body tag, which includes all the information we want the user to see on the page.

What is CSS?

If we consider HTML to give structure to a web page, then we can consider CSS to be how the web page then looks.

In CSS, we have a list of styles each of which corresponds to some part of the HTML page. Let's assume these names are called "nametags".

To make a CSS style, we use the following format: .nametag { style: styleType } So for example, we may have a tag like the following, which makes all the text in a section called "description" red. .description { color: red }

We can place the CSS into it's own file, or directly inside the HTML file within some "style" tags inside the "head" tags.
